# Getting Started

Welcome to your new Astro Paper blog!

Astro is a modern web framework that pulls your content from anywhere and serves it fast with less client-side JavaScript.

## Why Astro?

- **Zero JS Runtime:** Astro renders HTML on the server and strips away unused JavaScript.
- **Islands Architecture:** Only hydrate the components that need interactivity.
- **Content Collections:** Type-safe management of your Markdown and MDX content.
